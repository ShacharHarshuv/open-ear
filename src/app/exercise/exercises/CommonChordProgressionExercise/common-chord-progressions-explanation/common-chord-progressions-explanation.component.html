If you haven't already - check out the Chord Functions exercise for a basic introduction to chord progressions and Roman Analysis.

In this exercise, you can choose from a set of predefined progressions to practice.

The available progressions are one of the most popular chord progressions in popular music. If you think there is an important progression we missed, feel free to open an issue in Github or create a pull request

Tip! Try to think of songs you know that use each progression.
